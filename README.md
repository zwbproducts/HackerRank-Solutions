# HackerRank Solutions

A small collection of HackerRank problem solutions.

## Repository layout

- `solutions/c/` — C solutions (`.c`)
- `solutions/cpp/` — C++ solutions (`.cpp`)

## Build / run

Most programs read from stdin in the format HackerRank provides.

### C

```bash
gcc -O2 -Wall -Wextra -std=c11 solutions/c/<file>.c -o /tmp/a.out
/tmp/a.out < input.txt
```

### C++

```bash
g++ -O2 -Wall -Wextra -std=c++17 solutions/cpp/<file>.cpp -o /tmp/a.out
/tmp/a.out < input.txt
```
