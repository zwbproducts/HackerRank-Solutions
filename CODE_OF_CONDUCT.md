# Code of Conduct

## Our pledge

We want participation in this project to be a harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent.
- Keep feedback focused on the code.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Publishing private information.

## Enforcement

Project maintainers may remove, edit, or reject contributions that violate this code of conduct.
